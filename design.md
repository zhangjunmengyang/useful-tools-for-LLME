# TokenLab 设计规范

本文档定义了 TokenLab 应用的视觉设计系统，确保界面一致性、可读性和美观性。

---

## 🎨 设计原则

### 核心理念

1. **简约清爽** - 减少视觉噪音，突出内容本身
2. **高可读性** - 合理的字号层级，充足的对比度
3. **功能优先** - 设计服务于功能，不过度装饰
4. **一致性** - 统一的间距、字号、配色系统

### 设计风格

- **整体风格**: 现代极简主义，灵感来源于 GitHub 和 Linear
- **情感基调**: 专业、可信赖、高效
- **交互反馈**: 微妙但清晰的悬停和焦点状态

---

## 📐 字号系统 (Typography Scale)

采用 **1.25 倍模块缩放** (Major Third) 比例，基础单位 16px。

### 字号层级

| 层级 | 名称 | 大小 | 行高 | 用途 |
|------|------|------|------|------|
| **Display** | 页面标题 | 28px (1.75rem) | 1.3 | 模块主标题 |
| **H1** | 一级标题 | 24px (1.5rem) | 1.35 | 区块大标题 |
| **H2** | 二级标题 | 20px (1.25rem) | 1.4 | 小节标题 |
| **H3** | 三级标题 | 16px (1rem) | 1.4 | 卡片/列表标题 |
| **Body Large** | 正文大 | 15px | 1.6 | 重要段落 |
| **Body** | 正文 | 14px | 1.6 | 主要内容文本 |
| **Body Small** | 正文小 | 13px | 1.5 | 辅助信息 |
| **Caption** | 说明文字 | 12px | 1.4 | 标签、注释 |
| **Micro** | 微型文字 | 11px | 1.3 | 下标、极次要信息 |

### 字体家族

```css
/* 界面字体 - 主要文本 */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;

/* 代码字体 - 代码块、Token 显示 */
font-family: 'JetBrains Mono', 'SF Mono', 'Menlo', monospace;
```

### 字重规范

| 字重 | 值 | 用途 |
|------|-----|------|
| Regular | 400 | 正文、说明 |
| Medium | 500 | 标签、次级标题 |
| Semibold | 600 | 标题、强调 |

### 字号应用示例

- **模块标题** (Display): "分词编码"、"模型对比"
- **区块标题** (H1): "效率指标"、"分词结果"  
- **小节标题** (H2): "模型 A"、"模型 B"
- **组件标题** (H3): Expander 标题、Tab 文字
- **正文** (Body): 表单标签、选项文字
- **辅助文字** (Body Small): Metric 标签、帮助文字
- **微型文字** (Caption): Token ID 下标、状态标签

## 📏 间距系统 (Spacing)

采用 4px 基础单位的倍数系统：

```
xs    4px     0.25rem
sm    8px     0.5rem
md    12px    0.75rem
lg    16px    1rem
xl    24px    1.5rem
2xl   32px    2rem
3xl   48px    3rem
```

### 间距应用

- **组件内边距**: md (12px) 到 lg (16px)
- **卡片间距**: lg (16px)
- **区块间距**: xl (24px) 到 2xl (32px)
- **元素间最小间距**: xs (4px) 到 sm (8px)

---

## 🔲 圆角系统 (Border Radius)

```
none   0px      表格单元格
sm     4px      Token、小按钮、内联代码
md     6px      输入框、卡片、按钮
lg     8px      大卡片、容器
full   9999px   圆形元素
```

## 📦 组件规范

### 按钮

| 类型 | 背景 | 文字 | 边框 | 悬停 |
|------|------|------|------|------|
| Primary | #2563EB | #FFFFFF | none | #1D4ED8 |
| Secondary | #F3F4F6 | #111827 | #D1D5DB | #E5E7EB |
| Ghost | transparent | #111827 | none | #F3F4F6 |

**规格**: 字号 14px，内边距 8px 16px，圆角 6px

### 输入框

- **背景**: #FFFFFF
- **边框**: 1px solid #D1D5DB
- **聚焦边框**: #2563EB
- **聚焦阴影**: 0 0 0 3px rgba(37, 99, 235, 0.1)
- **圆角**: 6px
- **字号**: 14px (等宽字体用于代码输入)

### Metric 卡片

- **值字号**: 24px (1.5rem), JetBrains Mono, Semibold
- **标签字号**: 12px, Medium, 大写字母, 0.03em 间距
- **标签颜色**: #6B7280

### Token 块

- **字体**: JetBrains Mono
- **字号**: 13px
- **内边距**: 4px 8px
- **圆角**: 4px
- **下标 ID**: 11px, #6B7280

### Tab 标签

- **字号**: 14px
- **未选中颜色**: #6B7280
- **选中颜色**: #111827
- **选中指示器**: 2px bottom border, #2563EB

## ✨ 动效规范

### 过渡动画

```css
/* 快速过渡 - 按钮、悬停 */
transition: all 0.15s ease;

/* 标准过渡 - 展开、收起 */
transition: all 0.2s ease;
```

### 悬停效果

- **按钮**: 背景色变深/变浅
- **Token**: translateY(-1px) + 轻微阴影
- **链接**: 下划线

## 📝 文案规范

### 中文排版

- 中英文之间增加空格
- 数字与中文之间增加空格
- 使用全角标点符号

### 标签命名

- 简洁明了，避免冗余
- 动词用于按钮（如"加载"、"清除"）
- 名词用于标签（如"模型"、"结果"）
